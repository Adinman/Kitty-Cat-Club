## Local Deployment
First, make sure if you are using NodeJS v20.17.0+ before you locally deploy the project.
```
node -v :: Shouldn't be below v20.17.0!
git clone https://github.com/thekittycatclub/Kitty-Cat-Club
cd Kitty-Cat-Club
npm i
npm start
```
